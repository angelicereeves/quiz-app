Quiz App

This is a simple quiz application built with **React**. Users can input their name, 
select a category, answer a multiple-choice question, and see their results.

Features

- User-friendly form to start the quiz
- Multiple-choice questions with radio buttons
- Instant feedback after submitting an answer
- Option to try another question
- Responsive and styled with custom CSS
